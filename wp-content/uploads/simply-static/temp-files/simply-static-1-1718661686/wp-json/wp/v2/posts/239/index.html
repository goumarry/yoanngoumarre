      Description<\/h3>\n

      Ce projet consistait \u00e0 am\u00e9liorer une application de recherche de stage d\u00e9j\u00e0 existante. L’application comportait une interface web g\u00e9r\u00e9e avec Symfony et une application mobile d\u00e9velopp\u00e9e avec Android Studio. Les am\u00e9liorations apport\u00e9es visaient \u00e0 enrichir les fonctionnalit\u00e9s et \u00e0 optimiser l’exp\u00e9rience utilisateur. C\u00f4t\u00e9 admin, des fonctionnalit\u00e9s ont \u00e9t\u00e9 ajout\u00e9es pour g\u00e9rer les offres de stage, suivre l’\u00e9tat des candidatures, et administrer les utilisateurs. Pour les \u00e9tudiants, des fonctionnalit\u00e9s telles que la recherche d’offres, la candidature en ligne, et le suivi des candidatures ont \u00e9t\u00e9 am\u00e9lior\u00e9es. Ce projet a n\u00e9cessit\u00e9 une coordination efficace entre les \u00e9quipes travaillant sur les diff\u00e9rentes plateformes (web et mobile) et une compr\u00e9hension approfondie des besoins des utilisateurs finaux pour offrir une application plus intuitive et fonctionnelle.<\/p>\n
