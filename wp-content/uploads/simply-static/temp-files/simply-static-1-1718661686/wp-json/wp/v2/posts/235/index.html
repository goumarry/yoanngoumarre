      Description<\/h3>\n

      Ce projet consistait \u00e0 cr\u00e9er un site web de boutique en ligne en utilisant le framework PHP Symfony. Le site permettait de g\u00e9rer et de vendre des articles en ligne, incluant des fonctionnalit\u00e9s telles que la gestion des produits, des utilisateurs et des commandes. L’architecture MVC (Mod\u00e8le-Vue-Contr\u00f4leur) de Symfony a \u00e9t\u00e9 utilis\u00e9e pour structurer l’application, facilitant ainsi la s\u00e9paration des pr\u00e9occupations et rendant le code plus maintenable.
