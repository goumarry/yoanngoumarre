      Description<\/h3>\n

      Ce projet, d’une dur\u00e9e de trois semaines, impliquait une \u00e9quipe de sept personnes, dont deux sp\u00e9cialistes en r\u00e9seaux et serveurs. Nous avons cr\u00e9\u00e9 une application ludo-\u00e9ducative destin\u00e9e aux \u00e9l\u00e8ves de terminale en sp\u00e9cialit\u00e9 NSI (Num\u00e9rique et Sciences Informatiques). L’application comprenait divers mini-jeux visant \u00e0 enseigner les bases de l’informatique de mani\u00e8re interactive et engageante. J’ai principalement travaill\u00e9 sur le d\u00e9veloppement front-end en utilisant React, un framework JavaScript puissant et flexible. Pour la partie serveur, nous avons utilis\u00e9 Node.js, assurant une communication fluide entre le client et le serveur.
