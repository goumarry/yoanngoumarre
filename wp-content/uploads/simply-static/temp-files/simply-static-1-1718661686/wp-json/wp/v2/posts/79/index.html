      En tant qu’\u00e9quipe, nous avons collabor\u00e9 \u00e0 la conception compl\u00e8te d’une application de gestion d’\u00e9v\u00e9nements. Nous avons consacr\u00e9 une attention particuli\u00e8re \u00e0 d\u00e9finir la vision de l’application, en identifiant les besoins, en d\u00e9crivant les cas d’utilisation et en les classant par ordre de priorit\u00e9. Ensuite, nous avons programm\u00e9 l’application en utilisant le langage Java et d\u00e9velopp\u00e9 son interface graphique avec JavaFX.<\/p>\n
