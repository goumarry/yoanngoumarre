Développement d’une application serveur.

par goumarry | Juin 15, 2024 | BUT1

Whimsical, Java, JavaFX et Visual Paradigm

Objectif

Création d’une application serveur de gestion d’événements.

Compétences

  • Développer des interfaces utilisateurs.
  • Gestion d’un projet en équipe.
  • Connaissance Java, IHM, Visual Paradigm…

    Description

    En tant qu’équipe, nous avons collaboré à la conception complète d’une application de gestion d’événements. Nous avons consacré une attention particulière à définir la vision de l’application, en identifiant les besoins, en décrivant les cas d’utilisation et en les classant par ordre de priorité. Pour visualiser notre conception, nous avons élaboré des diagrammes de cas d’utilisation, des diagrammes de classes d’analyse métier et des diagrammes de classes d’objets. Ensuite, nous avons programmé l’application en utilisant le langage Java et développé son interface graphique avec JavaFX.

    Exploitation d’une base de données.

    PostgreSQL et R Studio

    Objectif

    Exploitation d’une base de données sur les accidents de la route.

    Compétences

    • Visualiser et comprendre des données.
    • Travail d’équipe.
    • Connaissance PostgreSQL et R Studio.

        Description

        En tant qu’équipe, nous avons collaborativement conçu, géré, administré et exploité les données d’une entreprise spécifique. Notre tâche principale était de mettre à jour et d’interroger une base de données relationnelle que nous avions préalablement conçue en respectant un cahier des charges détaillé. Une fois les données pertinentes extraites, nous avons utilisé le langage de programmation R pour créer des visualisations graphiques.           Recueil de besoins et découverte de l’environnement économique et écologique.

          par goumarry | Juin 15, 2024 | BUT1

          HTML/CSS, Word, Whimsical

          Objectif

          Consulter facilement des informations vulgarisées sur une ESN.

          Compétences

          • Appréhender les besoins du client et de l’utilisateur.
          • Travail d’équipe.
          • Identifier les acteurs et les différentes phases d’un cycle de développement.

            Description

            Dans le cadre de ce projet, nous avons entrepris de développer un nouveau site web à la fois sobre du point de vue écologique et visuellement attrayant, afin de présenter l’entreprise de manière optimale. Le site a été entièrement conçu en utilisant les langages HTML/CSS, permettant ainsi une personnalisation et une adaptation précises aux besoins de « Atos ». L’équipe a travaillé de manière organisée et agile, en utilisant des méthodes de gestion de projet efficaces pour garantir la réussite du projet dans les délais impartis.
